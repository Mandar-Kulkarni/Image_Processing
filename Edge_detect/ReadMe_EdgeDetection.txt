Steps for running the program:
1. Create a new empty project in visual studio.
2. Link OpenCV libraries to the project.
3. Copy the image "bird.jpg" to the same folder as the code. 
4. Add the EdgeDetection.cpp file in this folder to the project.
5. Run the code.
6. 4 output windows will be displayed with "Original Output", "Laplace edge map", "Sobel edge detection" and "Canny edge detection".
7. Press Escape to close the windows.
